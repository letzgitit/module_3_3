# git commit skill 사용법

## git 사용 순서
 - step 1. 해당 세션에서 작업한 파일들을 'git add' 
 - step 2. add한 파일들을 'git commit' 진행
    - commit 네이망과 커밋 메세지는 'git branch' 네이밍을 따른다.
 - step 3. commit한 파일들을 'git push' 
- step 4. `.claude/docs/progress.md`에 현재 작업 내용을 기록
    - [] 형태의 체크박스를 사용하여 직관적으로 작업 진행 상황을 기록한다.
